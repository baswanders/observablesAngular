Memory leak - no unsubscribe

In this example there are two subscribtions.

Pay attention to the counter, open the console log and see the counter there going in sync with the counter on the html.

Keep the console log open and go back to the home page.

Observe that the counter in the console log is still running!!
Next return to this page, the html counter looks fine, but in the console you see now TWO different values.

The reason for this is because we did not unsubscribe.
